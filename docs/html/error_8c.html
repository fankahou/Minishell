void debug ( char * msg)

Prints a debug message to stdout.

This function prints a debug message to the standard output (fd = 1). All messages are prepended with "\033[1;44m DEBUG \033[22;0m -> " and terminated with a newline. DEBUG must be defined as 1 for this function to execute, otherwise it will not do anything!
