Checks if a given character is a delimiter.

Checks if a character is the delimiter of a command. Some delimiters are made up of two characters and therefore require both values to be set so to be recognised properly!

  • ; = 1
  • && = 2
  • || = 3 if none of these are detected, 0 is returned.

◆ is_sym()

int is_sym ( char c1,
char c2 )

Checks which type a symbol is.

Checks for which type the given symbol is. The list of symbols and their return values are:

  • $ = 1
  • | = 2
  • << = 3
  • < = 4
  • > = 5
  • >> = 6
  • ; = 7
  • || = 8
  • && = 9

|| and && are only needed for the bonus. If no symbol is recognised, the function will return 0.

Parameters
c1First character of the symbol
c2Second character of the symbol
Returns
int
Return values
typeType of the symbol
